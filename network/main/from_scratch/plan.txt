4. Propagation arrière
    - we have
        - a list containing all the activation values of each neurone
        - the weighted sum of the each neurone of the final layer
        - the derivative of our cost function
        - the derivative of our activation function
    - the goal: calculate the gradient nabla C of each weight and bias

    a) the gradient of each param is stored in two list (for weights and bias):
        - the gradient weight list has the same shape as the weights list (idem for bias)
    b) create a list containing the errors
        - it is of the same shape as the biases
    c) calculate the final error (delta^L)
        - for each final weighted sum:
            - add to the elements of the last layer of the the error list: 
                derivate of cost function * derivative of the weighted sum
    d) propagate the final error to the previous layer, until the first one
        - start at index L (final layer index), and go down to 1 (including 1):
            - gradient C of l += delta l * transpose(activation of l-1)
            - gradient B of l += delta l
            - if not at layer 1: calculate error of the previous layer, delta l-1:
                delta l-1 = transpose(weight of l) * delta of l * hadamard of (sigma prime (z of l))

Repeat 3. and 4. to add up each gradient values

5. stochastic gradient descent:
    - goal: modify the weights and biases according to the calculated gradients
        - for each layer:
            step the weight: W -= learning rate/size of mini-batch times gradient of weights of layer l
            step the bias: B -= learning rate/size of mini-batch times gradient of biases of layer l
